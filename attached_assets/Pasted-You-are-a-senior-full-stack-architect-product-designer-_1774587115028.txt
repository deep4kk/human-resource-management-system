You are a senior full-stack architect, product designer, and systems engineer responsible for building a **production-grade, scalable, modular HRMS (Human Resource Management System)**.

Your goal is NOT to generate random code — but to design and build a **real-world, enterprise-ready system** with clean architecture, reusable modules, and extensibility.

---

# 🎯 CORE OBJECTIVE

Build a **complete HRMS platform** that is:

* Fully functional (no mock/stub logic unless specified)
* Modular (each module independently extendable)
* API-first (backend decoupled from frontend)
* Scalable (multi-company ready in future)
* Clean UI/UX (mobile-first responsive design)
* AI-ready (plug-and-play AI modules)

---

# 🧠 SYSTEM ARCHITECTURE

Follow this architecture strictly:

## Backend:

* Node.js (Express or NestJS preferred)
* REST API (GraphQL optional if needed)
* JWT-based authentication + role-based access control (RBAC)
* Modular service-based architecture

## Frontend:

* React (Next.js preferred)
* TailwindCSS + component system
* Fully responsive (mobile-first)
* Clean dashboards (admin + employee views)

## Database:

* PostgreSQL (preferred) OR MongoDB
* Proper schema design with relationships
* Audit logs for critical actions

## Integrations Layer:

* External APIs (biometric, payroll, OCR)
* Queue system (for async jobs like payroll, OCR, AI)

# 📦 MODULES TO BUILD

## 👨‍💼 1. CORE HR MODULE

### Features:

* Employee Profile Management

  * Personal, professional, salary, documents
* Department & Role Hierarchy

  * Dynamic tree structure
* Document Management

  * Upload, categorize, version control
  * OCR support (extract text from documents)

### Requirements:

* Dynamic schema (custom fields support)
* File storage (local/S3 compatible)
* Search & filter system

## 📅 2. ATTENDANCE & LEAVE MODULE

### Features:

* Biometric Integration:

  * Sync with eSSL devices or MS SQL DB
  * Scheduled sync jobs
* Real-time Attendance Logs
* Leave Management:

  * Apply, approve, reject
  * Leave balance auto-calculation
* Holiday & Shift Management

### Requirements:

* Rule engine (late marks, half days, overtime)
* Configurable policies

### Requirements:

* Approval workflows
